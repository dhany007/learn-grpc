# Hello

based on course: https://www.udemy.com/course/grpc-golang/

## Run
### Greet
for build greet:
```
  make build-greet
```
open terminal for server:
```
  ./bin/greet/server
```
open terminal for client:
```
  ./bin/greet/client
```
### Calculator
for build calculator:
```
  make build-calculator
```
open terminal for server:
```
  ./bin/calculator/server
```
open terminal for client:
```
  ./bin/calculator/client
```
